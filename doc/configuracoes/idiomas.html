O Zeanwork possui varios recursos que facilita no desenvolvimento de aplicações de multi-idiomas, para que o Zeanwork não fique carregando estes recursos em vão, você deve definir se irá utilizar estes recursos.
Para ativar/desativar estes recursos, você deve ir até o arquivo App/Configs/settings.php e localizar a configuração chamada useMultiLanguage.
O valor a ser definido é boolean (true/false), onde que true, será carregados os recursos automáticamente e false não será carregado.

Veja exemplo:
Configure::write('useMultiLanguage', true);


Do idioma

Para definir o seu idioma, você deve ir até o arquivo App/Configs/settings.php e localizar a configuração chamada fromLanguage.
O valor a ser definido é uma string.

Veja exemplo:
Configure::write('fromLanguage', 'pt-BR');
Para ver/acrescentar siglas de idiomas, você deve ir até o arquivo App/Configs/languages.php


Para o idioma

Para definir o idioma que será traduzido, você deve ir até o arquivo App/Configs/settings.php e localizar a configuração chamada toLanguage.
O valor a ser definido é uma string.

Veja exemplo:
Configure::write('toLanguage', 'en');
Para ver/acrescentar siglas de idiomas, você deve ir até o arquivo App/Configs/languages.php


Idiomas disponíveis para a tradução

Para definir os idiomas disponíveis para a tradução, você deve ir até o arquivo App/Configs/languages.php e localizar a configuração chamada languages.
