O helper form ajuda você a trabalhar com formulários.


Carregando o helper

Para você utilizar este helper, você prescisa definir que irá utiliza-lo. Veja mais em: Controllers > Carregando models, components, helpers e extensions.
O nome deste helper é form, este é o nome que você irá utilizar para carrega-lo.


Iniciando um form

start()
Este método inicia um novo form.

Parâmetros:
string $formName [opcional]
Texto a ser exibido no link.

array $options [opcional]
Opções e atributos para a tag form.
Veja algumas opções deste parâmetro:
string id
Id do form [opcional]

array action [opcional]
URL para o post.

array method [opcional]
Método do form (post/get).




Retorno:
string
String contendo o html.



Veja exemplo:
echo $this->form->start('users');



Finalizando o form

end()
Este método finaliza um form.

Retorno:
string
String contendo o html.



Veja exemplo:
echo $this->form->end();



Criando um input

input()
Este método monta input.

Parâmetros:
string $fieldName
Nome do input.

string $tag [opcional]
Nome da tag a ser utilizada.

array | string $options [opcional]
Opções e atributos para o input.
Se o valor informado for uma array ele interpretará como opções e atributos, mas se for uma string então será uma class de css.



Retorno:
string
String contendo o html.



Veja exemplo:
echo $this->form->input('username');



Algumas opções que o Zeanwork faz um tratamento special

string [type]
Define o tipo do input. Podem ser: text, checkbox, radio, password, hidden, submit, reset, button, file, image, textarea e select.

string [value]
Define um valor para o input.

string [id]
Define o id do input, caso não informado será utilizado o nome do input.

string [class]
Define a class css que irá utilizar.

string [label]
Define um label para este input.

boolean | string | array [divForLabel]
Define que o label deste input terá uma div somente para ela.
Se o valor informado for uma array ele interpretará como atributos para a div, mas se for uma string então será uma class de css.

boolean | string | array [divForInput]
Define que o input terá uma div somente para ele.
Se o valor informado for uma array ele interpretará como atributos para a div, mas se for uma string então será uma class de css.

Veja exemplo

echo $this->form->start('contact');

echo $this->form->input('name', array(
			  'label' => 'Nome:'
			, 'divForLabel' => array('style' => 'margin-top:5px;')
			, 'divForInput' => true
		)
	);

echo $this->form->input('email', array(
			  'label' => 'E-mail:'
			, 'divForLabel' => true
			, 'divForInput' => true
		)
	);

echo $this->form->textarea('mensagem', array(
			  'label' => 'Mensagem:'
			, 'divForLabel' => array('style' => 'margin-top:5px;')
			, 'divForInput' => true
		)
	);

echo $this->form->select('test', 
		array(
			  'options' => array(
					  array('optgroup' => true, 'label' => 'grupo')
						, array('value' => 'valor1', 'text' => 'option 1', 'disabled' => 'disabled')
						, array('value' => 'valor2', 'text' => 'option 2')
					, array('optgroup' => 'start', 'label' => 'grupo 2')
						, array('value' => 'valor3', 'text' => 'option 3', 'selected' => true)
					, array('optgroup' => 'end')
					, array('value' => 'valor4', 'text' => 'option 4')
					, array('value' => 'valor5', 'text' => 'option 5')
					)
			, 'class' => 'classCss'
			, 'label' => 'Teste de select'
			, 'divForLabel' => true
			, 'divForInput' => true
		)
	);

echo br(2) . $this->form->submit('submit', array('value' => 'Enviar', 'class' => 'submit'));
