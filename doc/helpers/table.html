O helper table ajuda você a trabalhar com tabelas.


Carregando o helper

Para você utilizar este helper, você prescisa definir que irá utiliza-lo. Veja mais em: Controllers > Carregando models, components, helpers e extensions.
O nome deste helper é table, este é o nome que você irá utilizar para carrega-lo.


Saiba mais

Para você saber mais sobre este helper, você deve ir até o arquivo Features/Helpers/table.php


Veja um exemplo de uso:

echo $this->table->start(900, 'idTabela', 0, 'classTabela');
echo $this->table->setClassTitle('classTitulo');
echo $this->table->setClassLine('classCssTr', 'classCssTr2');
echo $this->table->addColumn('Titulo da primeira coluna', '500px');
echo $this->table->addColumn('Titulo da segunda coluna', '400px');
echo $this->table->addLine('id_TR_1', 'coluna 1 - lina 1', 'coluna 2 - lina 1');
echo $this->table->addLine('id_TR_2', 'coluna 1 - lina 2', 'coluna 2 - lina 2');
echo $this->table->returnTable();
