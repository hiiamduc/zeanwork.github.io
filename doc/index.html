Ficamos gratos por você estar aqui para apreender mais sobre o desenvolvimento com o Zeanwork. Nesta documentação você terá a chance de apreender a desenvolver aplicações com grande produtividade e boas técnicas no desenvolvimento web. Também apreenderá sobre o design pattern MVC e á utilizar o Zeanwork da melhor forma possível.

Se você nunca utilizou um Framework PHP com MVC, você verá que não é tão complicado assim! O Zeanwork é perfeito para quem esta começando no mundo dos frameworks, mas é preciso saber um pouco sobre POO (Programação Orientada à Objetos). No desenvolvimento de uma aplicação, o Zeanwork vai lhe dando o caminho certo para que você possa prosseguir, com erros que lhe diz o que fazer para solucionar o seu problema.


Iniciando

Se você esta iniciando no mundo dos frameworks, já deve ter notado que existem muitos outros, então você pergunta: Porque mais um framework? Aqui temos uma resposta para esta pergunta! Conheça também nossas características.

Se você esta iniciando o desenvolvimento de aplicações com o Zeanwork, você pode ler nossas documentações e os tutoriais, ver os screencasts e trocar ideias com outros programadores que utilizam o Zeanwork.
Para que você possa desenvolver aplicações com o Zeanwork você deve saber programar em PHP e ter noções de POO (Programação Orientada à Objetos). Noções sobre o design pattern MVC também é desejável, mas se você não conhece, não tem problema, pois você irá aprender conosco.
