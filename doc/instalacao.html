O Zeanwork é extremamente simples de instalar, não há muito a ser feito.
O Zeanwork é compatível com o seu servidor! Você pode usar o Zeanwork em qualquer servidor, seja ele apache ou não.
Siga os passos abaixo para a instalação.

  1. Faça o download da última versão disponível no site oficial do Zeanwork.
  2. Descompacte o arquivo em uma pasta no seu computador.
  3. Copie esta pasta para o seu servidor local ou online.
  4. É isso ai! O Zeanwork deve estar funcionando perfeitamente.


Nota: O Zeanwork funciona perfeitamente em qualquer local do seu servidor, seja ele no root ou em uma pasta dentro do root.

Nota: Se você usa apache com o module rewrite habilitado, então você não precisa fazer nada, o Zeanwork sai funcionando perfeitamente. Mas se você não usa apache, então devemos alterar algumas configurações para a interpretação das URL's. Saiba mais em: Configurações > Protocolo de interpretação da URL
